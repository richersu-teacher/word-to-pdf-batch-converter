# 📄 word-to-pdf-batch-converter（課程計畫 Word 批次轉 PDF 工具）

這是一款專為學校處理「課程計畫上網公告」需求所設計的 **Word 批次轉 PDF 工具**。

每到課程計畫整理或公告前，常會有大量 Word 檔案分散在不同年級、領域、科目或教師資料夾中，  
檔案數量可能高達數十個甚至上百個。  
如果一個一個開啟 Word，再手動另存成 PDF，不但非常耗時，也容易發生漏轉、轉錯資料夾或檔名混亂的情況。

由於課程計畫上網公告時，附件通常建議使用 PDF 或 ODT 等較適合公開瀏覽與保存的格式，  
本工具主要協助將既有的 Word 課程計畫文件批次轉換為 PDF，  
減少人工重複操作，讓公告前的檔案整理流程更有效率。

除了課程計畫外，如果學校其他行政業務也有大量 Word 文件需要整理、公告或轉成 PDF，  
例如會議資料、行政表件、校內計畫、活動文件等，也可以使用本工具進行批次轉檔。  
只是本工具最主要節省時間的情境，仍是像課程計畫這類「檔案多、資料夾層級多、需要大量轉檔公告」的工作。

本工具透過電腦中已安裝的 **Microsoft Word** 進行轉檔，  
因此相較於使用其他第三方轉檔工具，較能維持原本 Word 文件中的表格、圖片、頁面配置與格式。

## ✨ 核心特色

### 📄 高度保留原始版面

本工具會呼叫電腦中已安裝的 Microsoft Word 進行轉檔，  
盡可能維持 Word 文件原本的字型、表格、圖片與版面配置。

適合處理對格式要求較高的文件，例如：

- 課程計畫
- 教案
- 會議資料
- 行政表件
- 校內計畫文件
- 學校常用 Word 表格文件

---

### 📂 支援大量文件批次轉檔

可以一次處理多個 Word 文件，  
不需要逐一開啟 Word、另存成 PDF。

特別適合處理：

- 多個年級資料夾
- 多個領域或科目資料夾
- 多位教師分別繳交的 Word 檔
- 多層資料夾中的大量文件

---

### 🛡️ 同名檔案處理

當輸出資料夾中已經有同名 PDF 時，工具提供：

- 覆蓋原檔
- 自動加序號另存新檔

避免重要檔案被誤覆蓋。

---

### 🧹 自動略過不必要檔案

工具會自動略過部分隱藏檔、系統檔或暫存檔，  
減少批次轉檔時的干擾。

## ⚠️ 注意事項

使用前請先閱讀以下說明：

1. **本工具不包含 Microsoft Word**

   本工具只是協助操作電腦中已安裝的 Microsoft Word。  
   使用者需自行安裝並合法授權 Microsoft Word。

2. **未安裝 Word 無法使用**

   如果電腦沒有安裝 Microsoft Word，本工具無法獨立完成轉檔。

3. **轉檔結果會受電腦環境影響**

   PDF 輸出結果可能受到以下因素影響：

   - Word 版本
   - 文件使用的字型
   - 文件內的圖片、表格或特殊物件
   - 電腦系統環境

4. **特殊字型請先安裝**

   如果 Word 文件使用特殊字型，建議在轉檔電腦上也安裝相同字型，  
   以避免 PDF 版面與原始文件不同。

5. **遇到防毒或 Windows 警告時**

   若防毒軟體、Windows defender 或校園資安系統出現提示，  
   請依照學校資訊安全規範處理。  
   若不確定是否能執行，建議洽詢學校資訊人員。

6. **建議先用少量檔案測試**

   第一次使用時，建議先用幾個測試檔案確認輸出結果，  
   確認沒有問題後，再進行大量批次轉檔。

7. **重要文件建議先備份**

   雖然工具提供同名檔案處理功能，  
   但處理大量正式文件前，仍建議先備份原始資料。

8. **轉檔期間請避免手動操作 Word**

   工具轉檔時會在背景呼叫 Microsoft Word。  
   建議轉檔期間不要同時手動開啟或操作大量 Word 文件，  
   以降低轉檔中斷或 Word 無回應的機率。

9. **大量檔案轉檔需要時間**

   若檔案數量很多，或文件內含大量圖片、表格，轉檔時間可能較長。  
   建議耐心等待工具完成處理。

## 🛠️ 開發說明

本工具使用 C# WinForms 開發，  
介面設計以簡單、穩定、容易操作為主。

轉檔時會呼叫使用者電腦中已安裝的 Microsoft Word，  
由 Word 本身負責將文件輸出為 PDF，  
因此較能保留原始 Word 文件的排版效果。

本工具不使用第三方排版引擎重新解析 Word 文件，  
主要目標是降低批次轉檔時的人工操作負擔，  
並盡可能維持原本文件在 Microsoft Word 中的顯示效果。
